- 发版时间: 2025 年 04 月 11 日
- 版本号: v2.2.0.0


## 新增
- 新增 统一认证 SSO 支持 OpenID Connect (OIDC)，并支持 Keycloak 作为 OIDC 服务提供者。
- 新增 统一认证 SSO 支持钉钉。
- 新增 统一认证 SSO 支持飞书。
- 新增 统一认证 SSO 支持企业微信。
- 新增 数据库对象授权支持到表级别，可对表进行 **DML**、**DDL**、**脱敏查看** 权限的配置。
- 新增 支持设置授权是否允许执行 **存储过程**、**存储函数** 的调用。
- 新增 在启用脱敏数据查看中，允许开发者临时 **申请脱敏数据查看** 权限。
- 新增 支持在线申请权限，并通过内置工单流程或者外置飞书、企业微信、飞书审批进行权限申请审批。
- 新增 SQL 类工单在审批结束后支持 **立即执行**、**定时执行**、**不执行** 三种执行策略。
- 新增 MySQL 语句支持调用存储过程、创建用户、创建角色、授权、删除用户/角色。
- 新增 PostgreSQL 语句支持调用存储过程、创建用户、创建角色、授权、删除用户/角色。
- 新增 DB2 语句调用存储过程。

## 优化
- 优化 钉钉、飞书、企业微信插件使用的资源可能处于脱管的情况。
- 优化 用户账号在授权环节的 UI 交互体验。

## 修复
- 修复 修复 DB2 for IBM i 兼容问题。
- 修复 修复 SQL 中 IN 语句解析不支持的问题。
