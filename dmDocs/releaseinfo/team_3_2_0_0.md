- 发版时间: 2025 年 11 月 20 日
- 版本号: v3.2.0.0

## 更新亮点
- 新增 **Redis** 数据源，并支持可视化操作、数据查询、SQL 审核、SQL 审计、CI/CD 等功能。
- 新增 **Hologres** 数据源，并支持可视化操作、数据查询、数据脱敏、SQL 审核、SQL 审计、CI/CD 等功能。


## 新增
- 支持 Redis **命令级授权**，并可通过查询规范限制具体命令的执行。
- 支持 Redis **160+** 命令，涵盖 String、Bit、Hash、List、Set、ZSet、HyperLog 多种数据结构。可一次执行多条命令。

## 优化
- 优化 对子账号授权时，隐藏未启用查询的数据源。
- 优化 对子账号授权时，在选择授权时间区域新增清空按钮。
- 优化 新建表可视化设计器中的增加/删除按钮，从顶部移至树节点内联显示。
- 优化 MySQL 创建存储函数时，增加 READS SQL DATA 选项。

## 修复
- 修复 开启规则后，子账号执行多条语句时，判断所需权限不准确的问题。
- 修复 环境配置权限判断错误的问题。
- 修复 结果集含有超大列时，导出 Excel 格式失败的问题。
- 修复 安全规范的范围列表下，删除范围时因缺失二次确认逻辑，导致无法强制删除的问题。
- 修复 在安全规范中，新建范围时，刷新实例报错的问题。
- 修复 环境描述超长导致报错的问题。
- 修复 Greenplum 在获取视图创建语句时只获取了视图部分，缺失创建部分的问题。
- 修复 GaussDB/OpenGauss/Greenplum 在创建序列时 SQL 无法解析的问题。
- 修复 DB2 创建视图语句不支持的问题。
