- 发版时间: 2025 年 02 月 12 日
- 版本号: v2.1.1.0


## 新增
- 新增 DB2 数据源支持。
- 新增 DB2 数据源对 29 个 SQL 审核规则的支持。
- 新增 DB2 数据源对 5 个 数据脱敏规则的支持。
- 新增 DB2 支持驱动选择，支持 DB2 for IBM i，之前的是 DB2 for IBM z/OS。
- 新增 SQL Server 数据源支持。
- 新增 SQL Server 数据源对 29 个 SQL 审核规则的支持。
- 新增 SQL Server 数据源对 5 个数据脱敏规则的支持。

## 优化
- 优化 数据源选项中的 Driver 选项可以编辑并被修改。
