- 发版时间: 2025 年 10 月 30 日
- 版本号: v3.0.0.0

## 更新亮点
- 新增 **StarRocks** 数据源，并支持可视化操作、数据查询、数据脱敏、SQL 审核、SQL 审计、CI/CD 等功能。
- 新增 **MaxCompute** 数据源，并支持可视化操作、数据查询、数据脱敏、SQL 审核、SQL 审计、CI/CD 等功能。
- 新增 **PolarDB for PostgreSQL** 数据源，并支持可视化操作、数据查询、数据脱敏、SQL 审核、SQL 审计、CI/CD 等功能。

## 新增
- 支持 StarRocks、Doris、SelectDB 展示外部库和外部表。
- 支持 Doris 可视化创建/编辑视图功能。
- 支持 AnalyticDB for MySQL 获取 Table、View 的 DDL 语句。
- 支持 AnalyticDB for MySQL explan 功能。
- 支持 手动方式添加阿里云数据源，包括 RDS（MySQL 和 PostgreSQL）、PolarDB（MySQL、PostgreSQL、PolarDB-X）、AnalyticDB for MySQL、MaxCompute。
- 支持 对 GaussDB/OpenGauss 已有表进行可视化表结构编辑。
- 支持 TiDB 数据源展示序列
- 支持 OceanBase for Oracle 查看表、视图、存储过程、存储函数的属性。
- 支持 OceanBase for MySQL 可视化编辑表。
- 支持 DB2 通过可视化交互方式删除物化视图。
- 支持 在数据库对象浏览器的不同对象层级上刷新，获取对象的最新状态。
- 支持 在 配置 > 个人资料 页面中新增安全选项卡，可以更新阿里云 AK/SK，方便获取数据源。
- 支持 结果集导出 JSON、INSERT。

## 优化
- 优化 对支持的数据源屏蔽数据库对象浏览器中不支持的右键菜单项。
- 优化 PostgreSQL/Greenplum/GaussDB/OpenGauss/PolarDB for PostgreSQL 的表结构设计器在新建和编辑状态下可以修改和指定 PK 名称。
- 优化 表结构设计器的可配置选项，精简输入选项，仅保留常用选项。
- 优化 PostgreSQL 可视化创建表不再支持创建临时表，因为临时表在 Session 关闭后会自动销毁。
- 优化 配置页面中修改规范名称和描述的弹窗交互，可拖动窗口。
- 优化 对于 DBLINK 的创建、删除操作权限调整为需要运维角色的权限。
- 优化 可视化交互设计器表单项自动勾选默认选项。
- 优化 隐藏 DB2 不支持的物化视图属性菜单选项。
- 优化 查询控制台引导页面中快捷按钮通过权限控制是否可点击。
- 优化 前端底层技术架构从 vue2.x 升级到 vue 3.x。

- 修复 Doris 创建别名函数解析出错的问题。
- 修复 账号被赋予全部资源权限后无法进行工单审批的问题。

- 修复 MySQL、OceanBase for MySQL、TiDB、MariaDB、PolarDB for MySQL 创建存储函数并指定类型参数的情况下，参数信息没有正确生成的问题。
- 修复 OceanBase for Oracle 无法以展开视图的方式查看视图列的问题。
- 修复 Greenplum 获取视图 DDL 语句错误的问题。

- 修复 RDS for PostgreSQL 展开表报错的问题。
