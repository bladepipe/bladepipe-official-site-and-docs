- 发版时间: 2024 年 9 月 15 日
- 版本号: v1.4.0.0

支持企业级 OpenLDAP 和 Windows 域用户登录。

## 新增
- 新增 LDAP 账号作为 CloudDM Team 的子账号进行登录。
- 新增 Windows Active Directory 域用户作为 CloudDM Team 的子账号进行登录。
- 新增 用户执行的 SQL 记录到控制台特定日志中以协助后续审计。
- 新增 MySQL show 系列语句全部支持。

## 优化
- 优化 登录流程在报错提示信息更清晰。
- 优化 脱敏规则中手机号规则描述文案中目前可以显示出具体号段参数。
- 优化 子账号使用DM过程中最小化权限，添加数据源、查询设置、安全管理、机器管理、工单系统。
- 优化 角色的权限分类体系使其和菜单具有一定关联性便于理解。
- 优化 使用统一风格展示系统设置菜单下：子账号管理、角色管理、环境管理、操作审计，四个页面。
- 优化 CloudDM Team Docker 版部署好后，重启计算机，现在 console、mysql、sidecar 会跟随 docker 重启而自动重启。
- 优化 子账号登录后，我的权限移动到右上角账户中心入口处。

## 修复
- 修复 用户登出时操作审计接口报空指针的问题。
- 修复 环境中没有开启查询的数据源时，点击环境展开图标会一直转圈的问题。
- 修复 某些情况下操作审计的操作资源内容为空的问题。
- 修复 当资源对象被删除后，操作审计中【操作资源】 显示为空的问题。
- 修复 CloudDM Team Docker 版，因日志配置错误产生 LOG_FILE_IS_UNDEFINED 文件的问题。
- 修复 CloudDM Team Docker 版，Docker 卸载脚本在 Ubuntu 24 上执行失败导致无法卸载的问题。
- 修复 机器管理 “查看配置文件” 不输入验证码，弹框文案没有国际化的问题。
- 修复 只有规则查看权限时可以看到 新增/编辑/修改 按钮的问题。
- 修复 当不具备查询设置权限时单具备其它管理权限时，点击查询设置会报没有权限的问题。
- 修复 子账号不具备工单发起权限，却有递交工单按钮的问题。
- 修复 DM && CC 共享 RDP 库权限设置相互覆盖的问题。
- 修复 CloudDM Team Docker 版机器管理页面，在下线机器后机器状态图标展示不正确的问题。
- 修复 个人中心在修改手机/邮箱后不能及时更新用户信息的问题。
- 修复 兼容旧数据升级后，因为老数据 bindType 为 null 导致不能修改密码的问题。
- 修复 子账号和环境页面中的表格，在环境列上样式错乱的问题。
- 修复 查询设置->集群管理-> 集群机器 页面中修改集群机器备注无效的问题。
- 修复 查询设置->集群管理-> 修改集群备注无效的问题。
