## 优化
- 优化 查询控制台中执行 SQL 允许一次执行多条 SQL。
- 优化 MySQL、Oracle 提交 SQL 工单时，若出现无法通过规则校验的 SQL，将会显示 SQL 行号。
- 优化 查询控制台日志面板右侧区，新增两个按钮可以跳转到控制台日志的最顶部和最底部。

## 修复
- 修复 查询控制台日志面板在执行 SQL 后无法定位到最后的问题。
