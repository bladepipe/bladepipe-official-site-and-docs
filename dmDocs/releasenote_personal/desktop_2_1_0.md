# 更新内容

## [新增]
- [新增] 新增 SQL SERVER 数据源支持。

- [新增] 新增 SelectDB 数据源支持。
- [新增] 新增 PolarDB-X 数据源支持。
- [新增] 新增 IBM Db2 数据源支持。
- [新增] 新增 Redis cluster、info 命令组的完整支持。
- [新增] 新增 Redis 账号/密码 方式链接。

- [修复] 修复系统语言非中文的环境下，字体显示错误的问题。
