- 发版时间: 2023年 09月 19日

# 更新内容

新版本的表结构编辑器支持更多的数据源。

![common w650_h400](../assets/personal/releaseinfo/desktop_2_1_3/v2_1_3.gif)

## [新增]
- [新增] 新表结构设计器支持 StarRocks 数据源。
- [新增] 新表结构设计器支持 Doris 数据源。
- [新增] 新表结构设计器支持 Oracle 数据源。
- [新增] 新表结构设计器支持 SQL Server 数据源。
- [新增] 新表结构设计器支持 ClickHouse 数据源。
- [新增] 启动迎宾图进度条展示。

## [优化]
- [优化] 本地存储的应用程序日志目录，软件会定期清理历史日志。

## [修复]
- [修复] IBM Db2 无法执行 SQL 的问题。
- [修复] 表编辑删除主键后的语句生成错误问题。
