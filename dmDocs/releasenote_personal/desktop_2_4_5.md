# 更新内容

## [新增]
- [新增] 达梦数据库查看同义词列表功能。
- [新增] IBM Db2、达梦 等数据库查看存储过程列表功能。
- [新增] IBM Db2、达梦、Greenplum、Doris 等数据库查看函数列表功能。
- [新增] IBM Db2、达梦、Greenplum 等数据库查看物化视图列表功能。
- [新增] IBM Db2、达梦、Greenplum 等数据库查看序列列表功能。
- [新增] IBM Db2、达梦、Greenplum 等数据库查看触发器列表功能。

## [修复]
- [修复] 连接 Oracle 数据库时，出现命令未正确结束的问题。
- [修复] IBM Db2 数据库编辑数据功能数据展示错误的问题。
- [修复] IBM Db2 数据库双击表时生成的 SQL 查询语句错误的问题。
- [修复] SQL Server 数据库使用子账号连接后，数据库数量不能正确展示的问题。
