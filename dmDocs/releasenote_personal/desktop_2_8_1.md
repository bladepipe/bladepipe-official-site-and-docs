- 发版时间: 2024年 8月 14日
- 版本号: v2.8.1

# 更新内容

## [新增]
- [新增] 添加数据源后，自动开启查询功能，无需手动再开启。
- [新增] 右下角后台任务列表增加刷新按钮，可随时更新列表信息。

## [优化]
- [优化] 删除数据源提示。
- [优化] MySQL 数据生成和压力测试功能性能，使用更稳定。
- [优化] 压力测试功能进度显示，目前进度与运行时间相关。
- [优化] Oracle 数据编辑功能，不支持的类型会禁用编辑并显示 `**UNSUPPROTED**`。

## [修复]
- [修复] 后台任务数据生成功能进度无法显示的问题。
- [修复] 后台任务终止后还存在列表缓存的问题。
- [修复] Oracle 数据源在数据对象浏览器上展开时报错的问题。
- [修复] 数据生成和压力测试功能获取日志信息有概率出现文字重叠的问题。
- [修复] 搜索框文字颜色提示侵入到其他页面的问题。
- [修复] 查询结果导出当前行功能点击无效的问题。
- [修复] 表编辑功能未做修改关闭会提示有修改的问题。
- [修复] 设计表等功能在执行完 SQL 语句后，窗口 UI 信息无法显示的问题。
- [修复] MySQL、PostgreSQL、SQL Server、Oracle 等数据源压力测试功能，全类型表无法使用的问题。
