# 更新内容

## [新增]
- [新增] 支持 ClickHouse 数据库，并支持驱动版本选择。
- [新增] 右键菜单可以快速新增数据源、创建/删除库、清空/删除表、查看表结构等。
- [新增] 数据源列表可以看到数据库、Schema、表的数量。
- [新增] 支持 Apple MacOS M1/M2 平台。

## [优化]
- [优化] 优化 UI 效果。

## [修复]
- [修复] 修复 PostgreSQL、Greenplum、MySQL 数据源某些 DDL 语句无法执行的问题。
- [修复] 修复 Mac OS 平台快捷键冲突的问题。
