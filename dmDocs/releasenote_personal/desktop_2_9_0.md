- 发版时间: 2024年 11月 11日
- 版本号: v2.9.0

# 本次亮点

- 全新的客户端架构，从 JCEF 方案升级为 Electron，操作体验更加顺滑。
- 为苹果 M 系列芯片而专门准备版本，相较优化前，使用 M2 Pro，启动速度从 30 秒缩短至 6 秒。

# 更新内容

## [新增]
- [新增] 窗口缩放 功能，通过菜单窗口可放大、缩小软件界面。
- [新增] 帮助菜单，可以快速获取产品的最新动态。
- [新增] universal 版本包下载，专为苹果 M 系列芯片架构优化。

## [优化]
- [优化] Linux 桌面安装包升级到 AppImage 格式，新格式只需双击即可运行。
- [优化] MacOS 系统中，点击窗口关闭按钮改为最小化方式，以符合大多数 Mac 软件操作习惯。
- [优化] 启动过程中多处耗时操作采用异步/并行 加载策略，软件平均启动速度提升 4～6 秒。
- [优化] 软件包体积，所有平台软件包体积平均减少 30 MB。

## [修复]
- [修复] Mac 桌面窗口在拖动改变大小时，出现窗口绘制区域错乱的问题。
- [修复] 某些页面中全选快捷键不生效的问题。
- [修复] MySQL 生成达梦数据库表结构时，语句之间缺少分号分割的问题。
- [修复] MySQL 生成 SQL Server 数据库表结构时，库名称出现 catalog 的问题。
