- 发版时间: 2024年 7月 30日
- 版本号: v2.8.0

# 更新内容

## [新增]
- [新增] 全新的添加数据源界面，与 Team 版本保持一致。
- [新增] 全新的事务功能，可以设置只读模式、隔离级别等。
- [新增] 查询设置功能，现在可以自由的开关数据源的查询功能。
- [新增] 全新的查询控制台，响应速度更快，右下角执行信息更详细。
- [新增] 数据库连接状态展示。
- [新增] 数据生成和压力测试功能中，选中表时的配置展示增加列配置信息展示。
- [新增] 数据生成和压力测试功能中，点击某个列的生成方式可定位到该列的配置页面。
- [新增] MySQL 数据源支持创建和修改视图、函数、触发器、存储过程等功能。
- [新增] Oracle  数据源支持创建和修改视图、触发器等功能。
- [新增] Oracle  数据源支持创建函数、存储过程等功能。
- [新增] PostgreSQL 数据源支持创建和修改视图功能。
- [新增] PostgreSQL 数据源支持创建触发器、函数、存储过程等功能。

## [优化]
- [优化] 软件启动速度缩减 10 秒。
- [优化] Schema 定位功能，定位逻辑由定位到选中的 Schema 变换为打开的 Schema。
- [优化] 查询控制台数据展示，滚动条 UI 展示更友好。
- [优化] 数据生成和压力测试功能初始化默认参数调整、优化文案显示。
- [优化] 数据生成和压力测试功能生成界面，可展示任务的状态，优化 UI 展示如字体样式等。
- [优化] 数据生成和压力测试功能日志获取界面，移除自动刷新，增加分页展示日志信息，按钮区分颜色等。
- [优化] 数据生成和压力测试功能预览界面增加文字提示，左侧表信息一览优化 UI 展示，右侧增加背景色区分增删改查。

## [修复]
- [修复] 软件丢失缓存问题，这个问题曾导致某些用户已有的缓存如数据源信息被清空。
- [修复] 编辑数据功能中，字段是否为 NULL 显示错误问题。
- [修复] Oracle 数据生成和压力测试功能，某些类型无法生成导致的报错问题。
- [修复] MySQL 编辑数据功能，在某些情况下出现的语句生成错误问题。
- [修复] PostgreSQL 数据源某些版本库表列信息展示错误的问题。
- [修复] PostgreSQL 数据源触发器、函数、存储过程等功能名称相同时，列表不显示的问题。
- [修复] PostgreSQL 数据源创建表时 serial 默认值设置问题，这个问题曾导致建表语句生成错误无法执行。
- [修复] IBM DB2 数据源查看列信息时，返回 DB2 SQL Error: SQLCODE=-313 错误问题。
- [修复] IBM DB2，Oracle，SQL SERVER 等数据源创建表时，可配置项类型与精度不匹配问题。
- [修复] 数据生成和压力测试功能，平均写入时间计算错误的问题，这个问题曾导致时间显示会出现负数。
- [修复] 展开函数和存储过程功能时，语言国际化设置错误问题。
