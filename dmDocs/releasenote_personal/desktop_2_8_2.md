- 发版时间: 2024年 9月 15日
- 版本号: v2.8.2

# 更新内容

## [新增]
- [新增] 数据源连接状态显示，并提供重新连接按钮。
- [新增] 数据源连接状态提示弹窗。

## [优化]
- [优化] 压力测试功能默认参数值调整为最佳性能。
- [优化] MySQL 数据源数据生成功能，数据写入速度平均可达到 5w/s 行。
- [优化] MySQL 数据源压力测试功能，数据随机写入速度平均可达到 2w/s 行。
- [优化] Oracle 数据源数据生成功能，数据写入速度平均可达到 2w/s 行。
- [优化] Oracle 数据源压力测试功能，数据随机写入速度平均可达到 1w/s 行。
- [优化] PostgreSQL、SQL Server、TiDB、OceanBase、PolarDB-X 等数据源的数据生成和压力测试功能，数据写入速度较旧版有明显提升。

## [修复]
- [修复] Host 地址修改后没有生效的问题。
- [修复] MySQL 数据源控制台执行 show create table 语句无反应的问题。
- [修复] Hana 数据源添加失败的问题。
- [修复] StarRocks、Doris 无法执行 create table 语句的错。
- [修复] 点击修改存储过程时，无法正常打开。
