- 发版时间: 2024年 3月 22日
- 版本号: v2.6.0

# 更新内容

## [新增]
- [新增] MySQL 等 10 种数据库的数据生成功能，用于生成测试数据。
- [新增] 数据生成功能支持多表多字段自由选择。
- [新增] 数据生成功能支持生成前系统配置，如自定义线程数，遇到错误继续，是否开启事务等。
- [新增] 数据生成功能支持自定义表生成行数。
- [新增] 数据生成功能支持字段生成器选项，可根据字段类型自由选择生成数字、字符串、时间等类型数据。
- [新增] 数据生成功能支持字段设置，如是否为空，为空的比例，最大最小值等。
- [新增] 数据生成功能支持进行生成前数据预览。
- [新增] 数据生成功能支持实时日志显示，包括数量与详细信息等。
- [新增] 以 Oracle 数据库为源端，对端支持 PostgreSQL 等 11 种数据库的 DDL 语句生成功能。
- [新增] 以 SQL Server 数据库为源端，对端支持达梦 等 6 种数据库的 DDL 语句生成功能。
- [新增] 以 IBM Db2 数据库为源端，对端支持达梦 等 4 种数据库的 DDL 语句生成功能。
- [新增] 以 TiDB 数据库为源端，对端支持 StarRocks 等 4 种数据库的 DDL 语句生成功能。
- [新增] 以 StarRocks 数据库为源端，对端支持 MySQL 等数据库的 DDL 语句生成功能。
- [新增] MySQL 数据库支持创建、修改和删除触发器功能。
- [新增] MySQL，达梦，SQL Server，IBM Db2，Oracle，PostgreSQL 等数据库删除触发器功能。
- [新增] 为支持视图和物化视图的数据库添加对应的双击生成模板语句功能。


## [优化]
- [优化] 快捷操作如清空、删除表高级选项的文字显示，描述更精确。
- [优化] 快捷操作如清空、删除表生成的 SQL 语句显示，与编辑器保持同样的 UI 效果。

## [修复]
- [修复] 序列列表、同义词列表无图标问题。
- [修复] 表列表在搜索后，展开列会重置搜索的问题。
- [修复] 展开数据源，在还在加载的时候右键点击删除，显示删除成功但列表中数据源仍然存在的问题。
