- 发版时间: 2024年 3月 5日

# 更新内容

## [新增]
- [新增] 以 MySQL 数据库为源端，对端支持 ClickHouse 等 11 种数据源的 DDL 语句生成功能。
- [新增] 以 PostgreSQL 数据库为源端，对端支持 SQL Server 等 10 种数据源的 DDL 语句生成功能。
- [新增] MySQL 数据库存储过程、函数、触发器双击生成语句模板的功能。
- [新增] MySQL 数据库查看存储过程参数列表的功能。
- [新增] MySQL 数据库查看函数参数列表、返回类型的功能。
- [新增] MySQL 数据库在 MySQL 库中查看常用系统内置函数的功能。
- [新增] StarRocks 数据库查看函数列表功能。

## [优化]
- [优化] 右键功能菜单顺序调整。
- [优化] 右键功能菜单，合并旧版本的生成和获取 DDL 功能，两者统一为 获取 DDL语句 功能。
