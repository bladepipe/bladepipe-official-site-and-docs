CloudDM Team 支持对添加的数据源开启查询功能，以及对数据源的各种参数进行配置。

## 启用查询功能 {#query}

1. 登录 CloudDM Team 控制台。
2. 点击 **查询设置**。
3. 选择数据源，点击数据管理的开关后。
4. 选择绑定的集群后，点击确认。
5. 返回数据源列表中，选择的数据源数据管理为 **开启状态** 则说明启用查询功能成功。

## 启用 CI/CD 功能 {#cicd}

1. 登录 CloudDM Team 控制台。
2. 点击 **查询设置**。
3. 选择数据源，点击 CI/CD 的开关决定这个数据源是否可以在 CI/CD 中使用。
4. 返回数据源列表，选择的数据源 CI/CD 为 **开启状态** 则说明启用成功。

## 修改参数设置

1. 登录 CloudDM Team 控制台。
2. 点击 **查询设置**。
3. 选择数据源，点击操作一栏的修改参数配置。
4. 在修改后的参数值中填入配置后点击生效配置。
5. 检查修改的参数是否正确，点击确认。
6. 刷新参数配置，若参数当前运行值为刚才配置的值，则说明 **修改参数设置成功**。
