本篇文章旨在介绍 CloudCanal 中申请码所包含的信息。

申请码是用于申请 CloudCanal 许可证的唯一凭证，包含了 **CloudCanal 版本**、**唯一标识** 以及 **用量信息**。

- **CloudCanal 版本**: 用于生成对应 CloudCanal 版本的许可证 
- **唯一标识**: 用于标识每一套 CloudCanal 系统，激活后自动生成
- **用量信息**: CloudCanal 已使用 **任务数** 和 **机器数**
