---

使用 OpenAPI 获取和添加阿里云的数据源以及创建数据迁移同步任务，需要设置阿里云 AccessKey ID 和 AccessKey Secret。该配置可随时清空或重新设置，确保云资源安全。

## 操作步骤

1. 点击 **配置** > **个人资料**。
2. 选择 **安全** 页签。
3. 在 **阿里云访问权限** 下，点击 **授权访问**。
4. 填入正确的阿里云 AccessKey ID (AK) 和 AccessKey Secret (SK)。
5. 点击 **确定**，配置成功。
   
   :::info
   若原本已授权过，则将覆盖原来的配置，更新成当前的配置。
   :::
6. 如需清空，点击 **解除访问权限** 即可。
