本文介绍机器相关的各类操作。

## 添加白名单
如机器变更过地址或者数据源的白名单发生过变化，需要重新将数据源添加进白名单，可以在 **数据源管理** 页面[按数据源添加](../../datasource_manage/ds_manage.md)，也可以在 **同步机器** 页面按照机器 IP 来添加。

若按机器添加白名单，系统会按照机器上绑定的数据任务相关的数据源，将这些数据源依次添加进该机器的白名单。具体操作步骤如下：
1. 选择需要添加白名单的机器，点击 **更多** > **检查白名单**。
2. 选择需要检查的 IP 类型，点击 **检查白名单**。
3. 如果没有需要添加的数据源，说明当前机器绑定的任务相关的数据源已添加过该机器的白名单，无需重复添加。   
   如果出现需要添加的数据源，点击 **更多** > **添加白名单**。
4. 选择需要添加的 IP 类型， 点击 **添加白名单**。
5. 该操作是一个[异步任务](../console_job_manage.md)，将在后台进行该操作。查看异步任务可点击 **更多** > **查看异步任务**。

## 更新公网 IP
如机器公网 IP 发生变化，可更新公网 IP。具体操作步骤如下：
1. 选择需要更新 IP 的机器，点击 **更多** > **更新公网 IP**。
2. 在对话框中输入新的公网 IP，点击 **确认**。

## 搜索绑定任务
支持在机器列表页按数据源搜索任务。具体操作步骤如下：
1. 点击页面右上角的 **搜索绑定任务**。
2. 选择需要查询的任务所关联的数据源，支持按 **源数据库** 和 **目标数据库** 搜索。点击 **查询**。
3. 列表将显示管理该数据源的任务。
