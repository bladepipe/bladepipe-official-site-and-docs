本文介绍如何通过 Kubernetes 安装 CloudCanal。

## 前置条件

- Kubernetes 已安装，可参考 [最小化 K8s 安装 (CentOS)](../dailyOP/minimal_k8s_for_centos.md) 文档。
- 以下端口未被占用。
  
   | 组件                   | 端口  | 用途                        |
   |--------------------|-------|----------------------------|
   | cloudcanal-mysql    | 32500 | 元数据库 MySQL 端口 |
   | cloudcanal-console  | 31111 | 用户控制台访问端口    |
   | cloudcanal-worker   | 32727 | Debug 任务端口 |
   | cloudcanal-prometheus | 31900 | 监控指标查询端口   |

## 步骤

1. (可选但推荐) 在 K8s master 节点生成 ssh-key，并打通各节点免登录（包括 master 节点自身）。
  
   ```bash
   ssh-keygen -t rsa -b 4096 -C "auto-login"
   
   ssh-copy-id user@remote_host1
   ssh-copy-id user@remote_host2
   ...
   ```

2. 运行以下指令安装。
   
   <InstallCommand 
     command="curl -fsSL https://tgzdownload.clougence.com/support/install_on_k8s.sh | bash -s -- <version>"
   />
