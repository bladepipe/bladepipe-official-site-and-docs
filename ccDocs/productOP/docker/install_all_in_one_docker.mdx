本文介绍如何通过 Docker 托管镜像安装 CloudCanal。

## 前置条件

- 已安装 Docker 和 [Docker Compose](https://docs.docker.com/compose/install/)，可参考 [最小化 Docker 安装](../dailyOP/minimal_docker_for_centos.md) 文档。 
- 以下端口未被占用。
  
   | 组件                    | 端口    | 用途            |
   |-----------------------|-------|---------------|
   | cloudcanal-mysql      | 25000 | 元数据库 MySQL 端口 |
   | cloudcanal-console    | 8111  | 用户控制台端口       |
   | cloudcanal-console    | 7007  | Sidecar 通信端口  |
   | cloudcanal-sidecar    | 18787 | 任务 Debug 端口   |
   | cloudcanal-prometheus | 19090 | 监控指标查询端口      |
