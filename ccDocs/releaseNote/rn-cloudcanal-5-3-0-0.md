## CloudCanal-5.3.0.0

发版时间:2025年11月28日 版本号: 5.3.0.0


## 亮点

- 支持 [MySQL -> MySQL/StarRocks/Doris/SelectDB/PostgreSQL 表达式表名](../operation/job_manage/create_job/create_regex_table_job.md)，可支撑海量表的迁移同步
- 新增 CloudCanal 控制台黑夜模式

## 新链路

- 开放 腾讯云 TDSQL MySQL -> TDSQL MySQL 全量同步/增量同步/数据校验/数据订正/结构迁移/DDL 同步（加列/减列/modify/rename/truncate）

## 新特性

- 支持 AWS RDS Oracle 全量迁移和增量同步
- 支持 Oracle/PostgreSQL/SQL Server 源端数据校验设置采样率（参数：checkSamplePercent ），无断点续传，支持叠加过滤条件
- 支持 MySQL -> PostgreSQL 多 Schema 迁移同步
- 支持 TiDB 源端多任务订阅相同表，对于超多 region 表显著降低拉取变更记录可能发生异常的概率
- 支持 MySQL/OceanBase for MySQL -> OceanBase for MySQL CHANGE COLUMN DDL 同步
- 支持 MySQL -> MySQL 地理信息类型同步
- 支持 在任务级别设置延迟告警 WebHook，方便拥有众多同步任务的用户对于关键任务进行单独监控
- 支持 钉钉、飞书、企业微信告警通知所有人的能力, 通过 配置 > 个人偏好 > CloudCanal > imAlertAtAll 参数设置
- 支持 语音告警通过代理发送方式，方便全内网部署用户使用该告警方式
- 支持 从迁移同步机器上传任务日志、机器日志到 S3 或 OSS (通过 配置 > 个人偏好 配置相关信息)，方便问题排查
- 支持 导出任务参数为 Excel 文件进行保存，方便问题排查
- 支持 全量任务性能/增量任务性能/机器监控图表图片导出
- 支持 修改订阅批量操作黑名单生效范围（已订阅表/新增表）
- 支持 已有任务参数保存为模板、应用参数模板到任务时，可进行预览
- 支持 按源和目标库表信息查任务相关信息的开放 API （/schema/listTransObjsByMeta）

## 优化

- 优化 任务 删除中 状态下，不能进行其他操作
- 优化 创建任务参数模版(创建、查询、删除)
- 优化 [模板配置项白名单规则](../operation/job_manage/job_op/param_template.md)
- 优化 创建任务筛选表的步骤中选择目标表的交互方式
