## CloudCanal-2.8.0.0

发版时间:2023年6月30日 版本号: 2.8.0.0

### 新链路

- 开放 PolarDbMySQL -> ADB for MySQL 链路，包括结构迁移、全量迁移、增量同步、DDL 同步（新增列/删除列）、数据校验

### 新特性

- 支持 MySQL/OceanBase 全量迁移数据过滤新增内存过滤开关

### 问题修复

- 修复 带有大数据量的表出现异常时，日志过大导致任务异常的问题
- 修复 TiDB 源端的 apply_commit.log 日志中，索引缺少的问题
- 修复 MySQL 为源端的增量同步，清空位点后找不到位点的问题
- 修复 StarRocks 全量迁移数据过滤功能不生效的问题
- 修复 回溯位点获取 bin_log 导致任务 OOM 的问题
- 修复 回溯位点设置时间后弹框不能关闭的问题
- 修复 全量迁移过程中停止任务，任务无法重启的问题
