## CloudCanal-2.3.4.0

发版时间:2023年2月28日 版本号: 2.3.4.0

### 新链路

- 暂无

### 新特性

- 支持 SQL Server -> MySQL DDL同步
- 支持 ClouGence 官网新增按天数购买许可证
- 支持 MySQL -> MySQL 精度类型制定P,M值
- 支持 Docker 基础镜像使用 OpenJDK1.8

### 优化

- 优化 CloudCanal 任务数判断逻辑
- 优化 ClouGence 官网个人中心的修改邮箱和手机号逻辑
- 优化 ClouGence 官网布局文案
- 优化 OceanBase 源端全量迁移捞取数据的方式

### 问题修复

- 修复 CloudCanal Schema 版本依赖问题
- 修复 MySQL -> Kafka Avro TimeStamp 报错问题
- 修复 SQL Server 源端数据排序规则大小写敏感问题
- 修复 修改订阅修改目标主键的配置被覆盖的问题
