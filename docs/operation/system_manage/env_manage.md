title: Manage Environment
description: It tells how to set up, configure and delete an environment.
---
BladePipe allows you to set up custom environments for DataSources to differentiate the business scenarios of the DataSources, which makes it easy for you to identify and manage the DataSources.

## Create an Environment
1. In the top navigation bar, click **Settings**.
2. In the left-side navigation pane, click **Environment** to enter the environment management page. 
3. Click **Create a new environment** in the upper right corner of the page and enter the environment name and description. 
4. Click **Add** to create the environment.

## Configure the Environment for a DataSource
1. In the top navigation bar, click **DataSource**.
2. Click **Add DataSource** in the upper right corner.
3. Select the desired **Environment** to configure the environment for the DataSource.

## Delete an Environment
1. In the top navigation bar, click **Settings**.
2. In the left-side navigation pane, click **Environment** to enter the environment management page.
3. Click **Delete** in the Operation column on the right side of the list, and click **Submit**.
:::tip
You cannot delete an environment if a DataSource is attached to that environment. Please delete the DataSource before you delete the environment.
:::
