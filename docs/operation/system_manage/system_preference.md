BladePipe supports advanced features such as setting preferences for DataJob creation and alert rule configuration. This page describes how to set system preferences and the supported parameter categories and usages.

## Configure System Preferences
1. In the top navigation bar, click **Settings**.
2. In the left-side navigation pane, click **Preference**.
3. Select the parameter to be modified and click the ![](../../assets/account_and_auth/4.png) or ![](../../assets/account_and_auth/3.png) icon in the **Modified Value** column to modify the parameter value. The parameter value range can be found in the **Reference** column.
    :::info
    If the parameter is in the **To create** state, it means the parameter has never been configured.
    :::
4. Click **Submit** in the upper right corner of the page.

## Parameter Categories

- **COMMON**：Configuration related to alert robots. For more information, see [Configure Alert Methods](../../productOP/byoc/maintenance/alarm_conf.md).
- **EMAIL_CONFIG**：Configuration related to the e-mail sending alert notifications. For more information, see [Configure Alert Methods](../../productOP/byoc/maintenance/alarm_conf.md).
- **USER_PREFER**：Parameter configuration related to DataJob creation. The configuration is applied to the created DataJobs by default.
- **SYS_ALERT**：Configuration related to Webhook alert notifications. For more information, see [Configure Alert Methods](../../productOP/byoc/maintenance/alarm_conf.md).
