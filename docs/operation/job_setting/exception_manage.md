title: View Exception Logs
---

BladePipe records all abnormal events generated by its core components. By viewing the abnormal event records of the abnormal monitoring, users can understand the overall operation of the system and easily troubleshoot problems.

## Procedure
In the top navigation bar, click **Sync Settings** > **Exception Log**.

- The Exception Tabs 
  - **Console**: Exceptions generated by console operations, including partial schema migration ability after task creation.
  - **Sidecar**: Exceptions generated by remote calls such as obtaining data source metadata during task creation, which will be forwarded to the Worker process by the console for execution. Relevant exceptions can be viewed if execution fails.
  - **DataJob**: Exceptions related to data task execution process.
