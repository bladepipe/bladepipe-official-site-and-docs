title: View Worker Logs
---
BladePipe allows you to view Worker logs in the BladePipe Cloud. The logs can help you understand the current situation of the running Workers.

## Procedure
1. In the top navigation bar, click **Sync Settings** > **Sync Worker**.
2. Select the corresponding cluster and click **Workers** in the Operation column.
3. On the right side of the page, click **Worker Log** to view the Worker log named sidecar.log, which records the main running log of the Worker.

:::info
Only the recent logs are shown in the BladePipe Cloud. If you request BladePipe staff to troubleshoot a problem, please provide the complete Worker logs.
:::
