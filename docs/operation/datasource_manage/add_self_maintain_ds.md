BBladePipe supports both **self-managed DataSources** and **cloud DataSources**. This article briefly describes how to add a self-managed DataSource to BladePipe.

## Supported DataSources
Currently, the supported DataSources include:     
AutoMQ, ClickHouse, Dameng, Db2, Doris, ElasticSearch, GaussDBForOpenGauss, Greenplum, Hana, Hive, Hudi, Kafka, Kudu, MariaDB, MongoDB, MySQL, OceanBase, Oracle, PolarDB-X, PostgreSQL, RabbitMQ, Redis, RocketMQ, SQL Server, SelectDB, StarRocks, TiDB, Tunnel.

## Procedure

1. Log in to the [BladePipe Cloud](https://cloud.bladepipe.com).
2. In the top navigation bar, click **DataSource**.
3. In the upper-right corner, click **Add DataSource**.
4. Select **Self-managed** for Deployment.
5. Choose DataSource type and enter details.
6. Click **Add DataSource** at the bottom of the page.
7. Go back to the DataSource list. The added DataSource is shown in the list.
