---
BladePipe supports both **self-managed DataSources** and **cloud DataSources**. This page introduces how to add a **AWS DataSource** to BladePipe.

## Supported DataSources
Currently, the supported DataSources include:     
Aurora (MySQL), Aurora (PostgreSQL), MariaDB, MySQL, Oracle, PostgreSQL, Redshift, Microsoft SQL Server.
## Procedure

1. Log in to the [BladePipe Cloud](https://cloud.bladepipe.com).
2. In the top navigation bar, click **DataSource**.
3. In the upper-right corner, click **Add DataSource**.
4. Select **Amazon AWS** for Deployment.
5. Choose DataSource type and enter details.
6. Click **Add DataSource** at the bottom of the page.
7. Go back to the DataSource list. The added DataSource is shown in the list.
