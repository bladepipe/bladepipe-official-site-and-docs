This page introduces how to activate BladePipe On-Premise.

## Terminology

- **Apply Code**: A unique code for applying for a license.
- **License**: A license is used to activate BladePipe On-Premise, including the authorization duration and number of authorized [links](../reference/service_difference.md).


## Install BladePipe On-Premise

1. Log in to [BladePipe](https://www.bladepipe.com?src=bp-doc).
2. Click **Try Community Free**, and install BladePipe following the instruction in [Install All-In-One (Docker)](../productOP/onPremise/installation/install_all_in_one_docker.mdx).
![](../assets/license/1.png)

## Get Apply Code

1. Log in to the BladePipe Console and click **Inactivated/Activated**.
2. An Apply Code is shown in the **Deployment Details** dialog. Click **Copy** to copy the code. 
![](../assets/license/2.png)

## Get a License
Log in to [BladePipe](https://www.bladepipe.com?src=bp-doc), and click **Get a License**.    
For on-premise deployment, BladePipe offers [Community, Enterprise Trial and Enterprise plans](../price/plans_diff.md). You can get the according license following the steps.
<Tabs groupId="license">
<TabItem value="community" label="Community/Trial License" default>
1. In Step 2, paste the Apply Code in the box.
![](../assets/license/3.png)
1. In Step 3, select Community Edition.
![](../assets/license/4.png)
1. Click **Proceed to Payment**.
2. In the License list, click **View** to check the license.
3. In the **View the License** dialog, click **Get Email Code**, and enter the code.
4. Click **Copy** to copy the license.
</TabItem>
<TabItem value="enterprise" label="Enterprise License">
1. In Step 2, paste the Apply Code in the box.
![](../assets/license/3.png)
1. In Step 3, select **Enterprise** Edition, and the number of links and validity period. For more details, please check [Pricing](../price/product_price.md). 
![](../assets/license/8.png)
1. Click **Proceed to Payment**.
2. In the License list, click **View** to check the license. 
3. In the **View the License** dialog, click **Get Email Code**, and enter the code.
4. Click **Copy** to copy the license.
</TabItem>
</Tabs>

## Activate BladePipe
1. Go back to BladePipe Console.
2. In the **Deployment Details** dialog, paste the license to the **Enter License** box, and click **Activate Now**.    
If BladePipe is in Activated status, you can click **Reactivate** and the **Enter License** box will appear.  
  ![](../assets/license/7.png)
    :::info
    You can click **Activated** to check the details of the license authorization.   
    Please renew your Community license before it expires, otherwise it will affect the running DataJobs.    
    For Enterprise license, please [contact us](https://www.bladepipe.com/about/) for renewal support or renew it by yourself before the license expires.
    :::
