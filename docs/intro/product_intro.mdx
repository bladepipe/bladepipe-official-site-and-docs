**BladePipe** is a real-time, end-to-end data integration platform that streamlines data flow across homogeneous and heterogeneous databases. 

Whether you need to build modern data pipelines, perform seamless data migrations, or implement CDC (Change Data Capture) for real-time synchronization, BladePipe provides a powerful, out-of-the-box solution. You can use its single, intuitive dashboard to manage your entire data replication lifecycle, from schema evolution to continuous monitoring.

![BladePipe Core Functions](../assets/product_intro/core_functions.png)

## Core Capabilities

### Data Migration (Batch ETL)

BladePipe migrates bulk data from your source to a target database. Built for high performance, it supports resumable transfers, parallel scanning, and batch writing to maximize throughput and minimize the impact on your operational systems. 

You can easily map metadata, apply custom data processing logic, and filter records on the fly. For complex architecture transitions, you can integrate Data Migration with **[Schema Evolution](#schema-evolution)** and **[Data Verification](#data-verification-and-correction)** to ensure strict data consistency.

### Real-Time Data Sync (CDC)

A robust CDC engine continuously consumes incremental logs from your source database and replicates operations to the target in near real-time. 

This continuous data pipeline supports dynamic DDL synchronization, event filtering, and custom code processing. It ensures highly efficient data writes, making it ideal for live analytics, caching, and real-time reporting.

### Schema Evolution

BladePipe intelligently migrates and synchronizes table schemas across different database dialects. 

It handles complex data type conversions and naming mappings automatically. You can run this capability independently or use it as an automated preparation step for your migration and synchronization pipelines.

### Data Verification and Correction

BladePipe performs precise field-by-field comparisons between your source and target data to identify discrepancies, and corrects inconsistencies automatically. 

You can run Data Verification as a standalone auditing tool or integrate it directly into your active data pipelines for continuous, automated quality assurance.
