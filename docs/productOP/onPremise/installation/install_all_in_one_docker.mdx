This page introduces how to install BladePipe using Docker.

## Prerequisite

- Docker and [Docker Compose](https://docs.docker.com/compose/install/) have been installed. Please refer to [Minimal Docker Installation](../maintenance/minimal_docker_for_centos.md) for guidance.
- The following ports are not in use.
  
   | Component                   | Port  | Use                                 |
   |--------------------|-------|-------------------------------------|
   | bladepipe-mysql    | 25000 | The port of metadata database MySQL |
   | bladepipe-console  | 8111  | The port of Web console for user    |
   | bladepipe-console  | 7007  | The port of Web console for Worker  |
   | bladepipe-worker   | 18787 | The port for debuggint DataJob      |
   | bladepipe-prometheus | 19090 | The port for prometheuse            |

## Procedure

1. Run the following command to download and execute the installer script.

   <InstallCommand

     projectHome="./bp_home"
   />
