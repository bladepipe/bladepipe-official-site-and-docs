For ease of use, BladePipe On-Premise sets a default username and password. If you need to modify it, please follow the steps below.

## Procedure

1. Log in to the BladePipe Console using the default username and password. 
2. Go to **Settings** > **Profile**. 
3. Click **Modify** next to **Password**, and enter the **current password** and the **new password**. 
4. Log in to the Console again.

## FAQ

**Q: What if I forget the modified password?**    
**A:** If you've logged in to the Console, just reset the password following the above steps. If you don't log in to the Console, please **install the Console again** or **seek BladePipe technical support** to reset the password.
