To improve user experience, BladePipe On-Premise sets the default verification code as 777777. If you want to use a more secure verification code, please follow the steps below to modify it.

## Procedure

Please refer to [Configure Console](https://www.bladepipe.com/docs/productOP/onPremise/maintenance/change_console_config/) to modify the parameter value **console.config.product.trial.verify_code=777777**
