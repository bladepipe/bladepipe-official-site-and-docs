By default, BladePipe provides built-in AccessKey and SecretKey for calling its open APIs. If you need to change these credentials, follow the steps below:

## Procedure
1. Log in to the BladePipe Console using the default username and password. 
2. Go to **Settings** > **Preference**. 
3. Click **Get AccessKey/SecretKey**. 
4. Click **Get Email Code** and fill in the code, and click **Submit**.
5. Log in to the Console again. Go to **Settings** > **Preference** > **Get AccessKey/SecretKey** to check if the keys are updated. 

## FAQ

**Q: What's the risk if I don't modify the default AccessKey and SecretKey?**     
**A:** These default username and password, verification code and AccessKey and SecretKey can be exploited to harm data security. For production environments, it is strongly recommended to update all default credentials and keys immediately after deployment.
