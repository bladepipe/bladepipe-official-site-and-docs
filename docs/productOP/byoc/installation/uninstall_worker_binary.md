---
This page describes how to remove a Worker of BladePipe from the system.

## Procedure
Run the following command to uninstall the BladePipe Worker in your local environment:
   ```shell
   /bin/bash -c "$(curl -fsSL https://download.bladepipe.com/binary/uninstall.sh)"
   ```
